# Mis Finanzas — versión web

Copia **pública y limpia** (sin datos personales) de la app de finanzas, para abrirla desde el móvil.

Es un único archivo estático (`index.html`). Los datos de cada persona viven solo en su navegador; en el móvil llegan **cifrados** desde su propio GitHub al activar la sincronización (Ajustes → Sincronizar). Nada personal se guarda en este repositorio.

## Publicar (GitHub Pages)

1. Sube este repo a GitHub como repositorio **público**.
2. En el repo: **Settings → Pages → Build and deployment → Source: Deploy from a branch → Branch: `main` / `/ (root)` → Save**.
3. En ~1 minuto estará en `https://<usuario>.github.io/mis-finanzas-web/`.
4. Ábrelo en el móvil y **"Añadir a la pantalla de inicio"** para tenerlo como una app.

> Herramienta informativa. No constituye asesoramiento financiero.
